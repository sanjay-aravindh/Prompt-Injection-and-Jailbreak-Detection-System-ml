# Prompt Injection and Jailbreak Detection System

A Machine Learning-based security system that detects **Prompt Injection** and **Jailbreak** attacks in Large Language Model (LLM) prompts. The project leverages Natural Language Processing (NLP) and traditional Machine Learning techniques to classify user prompts as **Safe** or **Malicious**, helping protect AI applications from adversarial prompt attacks.

---

## Overview

Prompt Injection and Jailbreak attacks are among the most significant security threats to Large Language Models (LLMs). This project develops a lightweight and efficient detection system using **TF-IDF feature extraction** and **Machine Learning classifiers** to identify malicious prompts before they reach the LLM.

The system preprocesses text prompts, extracts meaningful features, trains multiple classification models, and predicts whether a prompt is safe or malicious.

---

## Features

- Prompt Injection Detection
- Jailbreak Attack Detection
- Text Preprocessing Pipeline
- TF-IDF Feature Extraction
- Multiple Machine Learning Models
- Prompt Classification (Safe / Malicious)
- Model Performance Evaluation
- Confusion Matrix Visualization
- Easy-to-Extend Dataset Pipeline
